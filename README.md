# 📚 Solaris Scrolls — University Book Exchange Platform

A Java-based full-stack book exchange platform designed to help university students buy, sell, and exchange textbooks more affordably through a structured and user-friendly system.

---

## 🎯 Recruiter Summary

Solaris Scrolls is a full-stack software engineering project focused on building a scalable marketplace for student textbook exchange and transaction management.

The platform was designed to improve accessibility to affordable academic resources while providing a clean and reliable user experience.

This project demonstrates my ability to:
- Build full-stack applications using Java and SQL
- Design scalable software architectures
- Develop desktop-based user interfaces using JavaFX
- Work with relational databases and transaction workflows
- Apply software engineering design principles
- Collaborate in agile development environments
- Perform functional testing and debugging for reliability

---

## 🚀 Key Features

- 📖 Book listing and exchange platform for students  
- 👤 User account and transaction management  
- 💳 Secure SQL-backed transaction handling  
- 🔍 Search and browsing functionality for textbooks  
- 🖥️ Interactive JavaFX-based user interface  
- ⚡ Scalable and maintainable application architecture

## 📱 Problem → Solution

**Problem:**  
University students often face high textbook costs and lack an organized platform for exchanging academic resources affordably.

**Solution:**  
Solaris Scrolls provides a centralized book exchange platform that enables students to securely browse, exchange, and manage textbooks through a scalable full-stack application.
